# 🚀 AI_Career_Assistant

## 📖 Project Overview

AI Career Assistant is an intelligent web application that helps students and job seekers improve their career opportunities by analyzing resumes and comparing them with job descriptions.

The system calculates ATS scores, identifies missing skills, recommends suitable job roles, and generates a personalized learning roadmap.

## 🎯 Problem Statement

Many students and fresh graduates struggle to understand whether their resumes meet industry standards and job requirements.

This project solves that problem by:

- Evaluating resumes automatically
- Comparing resumes with job descriptions
- Identifying skill gaps
- Suggesting career improvements
- Recommending suitable jobs

## ✨ Features

### 📄 Resume Analysis
- Upload PDF or DOCX resumes
- Extract resume content automatically

### 🎯 ATS Score Calculation
- Evaluate resume quality
- Generate ATS compatibility score

### 🔍 Job Description Matching
- Compare resume with job description
- Calculate matching percentage

### 🛠 Skill Gap Analysis
- Detect existing skills
- Identify missing skills

### 💼 Job Recommendations
- Suggest relevant job roles

### 📚 Learning Roadmap
- Recommend skills and technologies to learn
